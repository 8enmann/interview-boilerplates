# Hello World

## Install

  * Install Elixir
  * Install dependencies with `$ mix deps.get`

## Use

Start Phoenix server with `mix phx.server`.

Now you can visit [`localhost:4000/hello`](http://localhost:4000/hello) from your browser.

## Test

Run your tests with `mix test`
